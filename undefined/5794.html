termux如何搭建hexo

发表于2022-08-22|更新于2022-08-22
|字数总计:758|阅读时长:2分钟|阅读量:

ok 啊 刚才那个有点问题 然后github那边太卡 删不掉 所以重发一遍
首先先安装 termux 这是一个可以模拟 linux 的软件
安装好了之后我们进行安装工具
首先更换下载源
这里我用的是清华源
“sed -i ‘s@^(deb.*stable main)$@#\1\ndeb https://mirrors.tuna.tsinghua.edu.cn/termux/termux-packages-24 stable main@’ $PREFIX/etc/apt/sources.list”
不要复制括号
然后我们更新软件
pkg install update

接着安装 nodejs 脚本 git 和 vim
pkg install nodejs-lts
pkg install git
pkg install vim

安装好之后我们安装 hexo
npm install hexo-cli -g
安装好之后创建 hexo 目录
mkdir hexo
然后我们给它所有权限防止出错
chmod 777 hexo/
然后切换到 hexo 目录
cd hexo
接着初始化 hexo
hexo init
查看 hexo 是否安装成功
hexo v
如果出现一大串数字加字母就代表成功了 (就是我懒懒得吧图片放上了) 然后启动 hexo 测试
hexo g 生产静态界面
hexo s 在本地端进行链接测试
复制他给的地址打开 如果出现网页了 恭喜你 hexo 已经安装好了
然后我们来安装 butterfly 的主题 (个人推荐 因为作者是国人 官网也有中文教程)
先点 CTRL 加 C 停止服务
输入下面的指令安装 butterfly
git clone -b master https://github.com/jerryc127/hexo-theme-butterfly.git themes/butterfly

不行就多试几遍 换网
然后我们打开配置文件改一下
vim _config.yml
在最后几行里有一个名为”theme” 的
把后面的”landscape” 改为”butterfly”
(要点击 i 才能编辑 vim)
好了之后点 ESC 输入”:wq” 退出
后面安装渲染
npm intall hexo-renderer-pug hexo-r
再次重新生成界面并且开启本地服务测试
hexo g
hexo s
成功的话你就会发现网页的主题已经变了
然后我们继续打开_comfig.yml
找到 #Site
把 langguage 里面的”en” 改为”zh-CN”
上面的自行修改就可以了 具体可以参考视频 https:://b23.tv/WgjU45M
然后我们来修改网站背景和封面还有头像
首先输入：我们去找一个图床网站 网址我放在下面了
https://imgloc.com/
上传喜欢的图片 复制链接
然后输入:vim themes/butterfly/_config.yml 注意！ 这里是主题文件的 config 不要搞混
输入”/top” 找到 index.img 点击回车
把刚才的链接复制进去 (注意 要加一个空格)
背景：先上传图片 复制链接 再到 vim themes/butterfly/_config.yml 输入 / 背景 找到后点击回车
在 backgrond: 后面加一个空格 输入 url (你的图片链接)
头像也一样 和上面一样 但是搜索要搜 / 头像
然后复制就去就行
注意！ 头像最好是圆形的

然后就没什么了
部署到 github 上的我会明后天发
具体美化可以上 butterfly 官网去看
https://butterfly.js.org
就到这了
再会各位
